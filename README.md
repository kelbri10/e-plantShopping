# e-PlantShopping

e-PlantShopping is an E-commerce applicatioin completed as a final project submission as part of the IBM Full Stack Developer Certification on Coursera.

## Technologies Used 
- ReactJS
- Redux
- Vite
- CSS

## Functionality
Using e-PlantShopping, you are able to: 
- [x] Add Plants to Cart 
- [x] Remove Plants From Cart
- [x] View Total Cost
- [x] View Plant Catalog by Category
- [x] View Quantity In Cart while Browsing
