# e-PlantShopping

e-PlantShopping is an E-commerce applicatioin completed as a final project submission as part of the IBM Full Stack Developer Certification on Coursera.

## Technologies Used 
- ReactJS
- Redux
- Vite
- CSS

## Functionality
Using e-PlantShopping, you are able to: 
    - Add Plants to Cart 
    - Remove Plants From Cart
    - View Total Cost
    - View Plant Catalog by Category
    - View Quantity In Cart while Browsing
